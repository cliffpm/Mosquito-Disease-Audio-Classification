# Mosquito-Disease-Audio-Classification
Deployed a neural network trained to classify between mosquito species that carry harmful diseases versus other noises (background, harmless mosquito species, etc.)

# Instructions to try it yourself!
1. First make sure Arduino and Edge Impulse CLI is downloaded properly
2. Unzip the Edge Impulse zip file in this repository and extract. Run the following bat file according to the OS your device is running on
3. After the bat file has finished executing, open a new terminal and run `edge-impulse-run-impulse` and the sampling and prediction should begin and repeat periodically!
